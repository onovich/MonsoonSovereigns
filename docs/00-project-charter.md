# 00. 项目章程

**状态：FROZEN**

## 1. 一句话定义

《季风诸王》是一款以 16 世纪东南亚大陆为舞台的单机、可暂停实时大战略游戏。玩家通过宫廷任命、臣属义务、人口动员、季风战役与战后再分配建立霸权；土地占领只是控制的一部分，真正的统治取决于人、路线、信用与代理人。

## 2. 产品愿景

目标不是给《信长之野望》换一张陌生地图，而是抓住它最有价值的组织体验，并用更适合该地区的政治结构重构：

```text
人口与生产
→ 任命、分封与臣属义务
→ 备战、运输与季节窗口
→ 会战、围城与劝降
→ 人质、贡赋、赏罚与重新安排权力
→ 更大的网络、更高的内部风险
```

玩家应持续面对以下问题：

- 哪位地方统治者值得保留，哪位必须替换？
- 给一名功臣更多收入权，能否换来短期胜利，却制造长期威胁？
- 雨季前是否还有时间围城？
- 该直接控制、设为臣属、索取贡赋，还是只要求军事通道？
- 继承人是否能维持前任依靠个人威望建立的网络？

## 3. 设计支柱

### P1. 权力必须通过人来执行

玩家没有“直接管理所有地区”的超能力。地方发展、征发、调略、备战与外交承诺由具体人物和机构执行。能力、忠诚、利益、距离和信息都会改变结果。

### P2. 臣服不等于吞并

一个地方可以有唯一实际控制者，但同时存在宗主、贡赋、出兵、人质、贸易与宣称关系。玩家可以取得霸权而不直接拥有每个地区；大帝国也可能在继承时迅速瓦解。

### P3. 战争从农业和日历开始

军队不是从库存中瞬间生成。动员会抽离劳力，行军依赖道路、河流、仓储与役畜，季风改变通行与疾病风险，战争结束后还会留下人口、财政和政治后果。

### P4. 人口比空白土地重要

经济和军事建立在家庭、劳役、工匠、随从、商人和地方网络之上。关键战争目标往往是首都、人口、交通、朝贡和象征性承认，而不是每一片连续领土。

### P5. 信息应可解释，但不完全

玩家能看见重要判断依据、估计区间和主要修正，不能直接读取所有真实世界状态。AI 和玩家都依据各自情报行动；“为什么没出兵”必须能给出具体原因。

### P6. 扩张产生结构性风险

不使用粗暴橡皮筋平衡。通过通信距离、代理人坐大、继承、地方利益、人口恢复、盟友恐惧和占领整合，让规模自然产生成本。

## 4. 目标玩家

- 喜欢《信长之野望》《三国志》《十字军之王》《欧陆风云》《全面战争》战略层，但希望减少重复微操的玩家。
- 喜欢角色关系与制度，但不希望游戏退化为纯事件小说的玩家。
- 接受复杂系统，要求界面清晰、结果可追溯的 PC 玩家。
- 对较少被游戏表现的东南亚历史感兴趣，但不要求预先熟悉人物。

## 5. 产品体验目标

- 前期：资源有限，依赖少数可信代理人，理解本地网络。
- 中期：跨区域战役、臣属谈判、人才与财政结构选择。
- 后期：重点从逐城操作转为维持霸权、继承、军团和地方利益平衡。
- 一局可以通过多种目标结束，不强迫把整张地图变成单一颜色。

## 6. 一级平台

- Windows 10/11 Electron 桌面版。
- Windows/macOS 上的 Chrome/Edge Web 版；macOS Web 为兼容目标而非完整 QA 承诺。

二级平台：Firefox。暂缓：macOS 桌面 app。非首发：Safari、移动浏览器、主机。

## 7. 1.0 目标范围

- 一张精细但受控的东南亚大陆地图。
- 1531、1551、1569、1581 中至少三个可玩剧本；最终数量由内容产能决定。
- 约 12—16 个主要可玩势力，若干小型自治政治体。
- 约 80 个重要城市/王都/要塞中心，400 个左右地区节点。
- 约 600 名有来源的历史人物与适量程序生成随从；具体数量不是质量目标。
- 完整臣属、经济、动员、战役、围城、外交、继承、家臣提案和 AI 系统。
- 手动节点式会战是 TARGET，不是无条件首发承诺。

## 8. 非目标

- 不做全 3D 单兵战场或数万士兵实体模拟。
- 不做角色生活模拟、开放世界或动作战斗。
- 不做现代国族叙事的“缅甸对泰国”等单线冲突。
- 不做所有东南亚地区的百科式覆盖。
- 不以数千个按钮和资源种类制造深度。
- 不在核心闭环成立前开发联机、DLC 框架、复杂海战或脚本 Mod。
- 不保证每个历史争议都有唯一答案；游戏需要展示不确定性和建模边界。

## 9. 成功指标

### 产品指标

- 玩家能用自己的话解释臣属、实际控制和宣称的区别。
- 玩家在战役前会主动查看季节、补给、盟军和农业代价。
- 后期主要操作是政策、任命、目标和例外处理，而不是逐地区点击。
- 不同统治结构能够产生可辨识的势力风格。

### 技术指标

- 浏览器与 Node Runner 确定性一致。
- 压力场景在目标硬件上无持续主线程卡顿。
- 存档可版本迁移、校验和恢复。
- 核心系统可通过无界面跑局和不变量测试验证。

### 文化指标

- 主要势力不被简化为现代民族标签。
- 历史暴力有后果和语境，不作为纯粹正反馈。
- 正式内容的关键事实和视觉选择可追溯到来源或顾问审查。
