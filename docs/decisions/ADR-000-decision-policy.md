# ADR-000：架构决定记录政策

- 状态：Accepted
- 日期：2026-06-23

## 规则

满足任一条件的变化必须新增 ADR：

- 改变一级平台、语言、渲染器、桌面壳或模拟线程模型。
- 新增重量级生产依赖或运行时服务。
- 改变领域模型的核心所有权、臣属、时间或确定性语义。
- 改变存档兼容策略、命令协议、Mod 安全边界。
- 推翻 `PROJECT_STATUS.md` 中 FROZEN 决定。

ADR 必须包含：上下文、决定、备选方案、后果、迁移/回滚、验证证据。只有 `lead_orchestrator` 与 `systems_architect` 均接受后才能改为 Accepted。
