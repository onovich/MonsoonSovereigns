# 01. 游戏设计总文档（GDD）

**状态：核心 FROZEN；具体参数 TARGET**

## 1. 产品概念

### 工作标题

- 中文：**季风诸王**
- 英文工作名：**Monsoon Sovereigns**
- 内部代号：**Project Mandala**

名称尚未做商标检索，禁止在代码外部把工作标题当作正式商业名称。

### 高概念

一款以 1531—1600 年东南亚大陆为背景的可暂停实时大战略游戏。玩家经营的不是一个边界固定的现代国家，而是一套围绕宫廷、王族、地方首领、城市、寺院、商路和人口义务组成的权力网络。胜利来自让关键统治者承认你的地位，并让这种承认在战争、距离和继承中持续。

### 玩家承诺

玩家最终应能够说：

> “我赢下这场战争，不是因为兵更多，而是因为提前两季储粮、让一个不可靠的臣属交出人质、把可信任的将领放在渡口，并在敌方继承危机时迫使三个地方首领倒向我。”

## 2. 核心循环

```mermaid
flowchart LR
  A[观察人口、财政、关系和季节] --> B[任命、授权、谈判与投资]
  B --> C[代理人执行开发、征发、调略和备战]
  C --> D[选择战役窗口与政治目标]
  D --> E[行军、补给、会战、围城和劝降]
  E --> F[贡赋、人质、赏罚、迁徙与驻军安排]
  F --> G[臣属网络扩大但内部负荷上升]
  G --> A
```

### 每月循环

1. 宫廷评议：财政、粮食、威胁、提案、任命、外交承诺。
2. 设定方向：地方方针、军团目标、战役准备、贸易与政策。
3. 时间推进：代理人行动、市场与人口变化、情报更新、军队移动。
4. 重大决策暂停：战争宣告、倒戈、继承、围城突破、投降与赏罚。
5. 月末结算：薪俸、贡赋、忠诚、政治负荷和 AI 重规划。

### 年度循环

- 收获与缺粮。
- 朝贡与礼仪承认。
- 官职、继承人和婚姻安排。
- 兵役、劳役和人口恢复。
- 季风改变交通与出征窗口。
- 势力战略重新评估。

## 3. 游戏对象与玩家层级

### 玩家身份

玩家控制“统治宫廷”，通常由在位统治者代表。玩家并不直接扮演每位人物；统治者死亡后继续控制其合法继承的宫廷或被迫重组的继承政权。

### 权力层级

```text
统治者 / 宫廷
  ├─ 中央官职与直属随从
  ├─ 直辖城市、仓库和军队
  ├─ 地方总督 / 城主
  │    └─ 地区首领、村社、寺院与商人网络
  └─ 臣属政治体
       ├─ 自身宫廷与继承人
       ├─ 自有收入和军队
       └─ 对宗主承担义务包
```

玩家通过命令、任命、承诺和资源改变代理人的可行行动，不把每个地区当成可直接操纵的生产格。

## 4. 地图和空间

### 地图表达

底层是图结构，视觉上为 2D/2.5D 区域地图：

- **政治中心**：王都、城市、要塞、港口、关隘。
- **地区**：人口、农业、森林、矿产、工匠和地方服从的基本单位。
- **路线边**：道路、山道、河道和沿海航线。
- **特殊节点**：渡口、仓库、市场、寺院中心、象源或关键山口。

### 空间不是现代国界

- 每个地区有唯一实际控制者，保证结算明确。
- 可有宗主链、贡赋、军事通行、宣称和影响。
- 地图颜色默认显示“实际控制”，可切换到“承认关系”“贡赋”“贸易”“宗教网络”“情报可信度”等模式。
- 未有效控制的边缘地区不应自动填满势力颜色。

## 5. 时间、季节与节奏

### 时间模型

- 可暂停实时。
- 权威逻辑以“游戏日”为固定步长。
- 玩家可选择暂停、1x、2x、4x、最高速度。
- 重大事件和可配置警报自动暂停。

### 季节模型

不同气候区使用各自月历，不用一条全地图统一“雨季 -30%”：

- 凉干季：远征和道路运输较稳定。
- 热季：水源、疾病和役畜压力上升。
- 西南季风：部分道路泥泞，河运增强或洪水风险上升，围城与疾病条件改变。
- 转换期：天气不确定，窗口风险提高。

季节影响必须通过可解释的路线、运输、疾病、农时和河流水位表现，而不是纯属性 debuff。

## 6. 统治、臣属与义务

### 关系类型

- 独立。
- 朝贡/承认宗主。
- 臣属：承担明确的贡赋和军事义务。
- 附庸性总督：高度依赖宗主但保留地方政权。
- 占领：军事控制尚未转化为稳定统治。
- 争议：实际控制明确，但存在强宣称或地方抵抗。

### 义务包

每段宗主—臣属关系由可谈判义务组成：

```text
年度贡物 / 金钱 / 粮食
战时兵额或象只
通行与仓储权
人质
婚姻
礼仪出席
不得独立外交
贸易或市场权
边境防务责任
```

义务越重，短期收益越高，违约与叛离风险也越高。

### 服从的来源

- 宗主军事实力与近期胜利。
- 合法性、宗教赞助和礼仪地位。
- 个人关系、婚姻、人质和恩赏。
- 共同敌人、贸易依赖与安全需求。
- 宗主是否履行保护和承诺。
- 距离、地方实力、继承危机和怨恨。

### 帝国瓦解

霸权不能只用一个“稳定度”数字表示。统治者死亡或大败时，每段臣属关系独立重估：

```text
新统治者合法性
旧统治者个人关系是否可继承
人质是否有效
地方军事实力
邻近竞争宗主
拖欠的赏赐与保护
履约历史
```

因此庞大网络可能部分延续、重新谈判，或连锁解体。

## 7. 人物、宫廷和代理人

### 公开能力

建议四项主能力，0—100 显示、内部 0—10000：

- 统率：组织、协调、士气与行军。
- 武勇：局部战斗、威慑和个人风险。
- 政务：税赋、开发、组织与行政。
- 智略：情报、调略、外交和风险判断。

### 人物差异的主要来源

- 性格：谨慎、冒进、务实、虔敬、奢侈、残酷、宽厚等。
- 利益：家族、所领、贸易、寺院关系、继承位置。
- 关系：亲属、婚姻、恩主、仇怨、同僚竞争。
- 忠诚、野心、声望、健康和年龄。
- 身份与资格：王族、地方世袭首领、宫廷官、商人代理、宗教人物。
- 信息与偏见：人物不会全知，也不永远提出最优方案。

### 任命

职务和政策归属于机构，不因现任人物死亡而凭空消失；人物影响执行能力、优先级和偏差。玩家需要在“能力强但危险”与“忠诚但平庸”之间选择。

### 家臣提案

提案由机会触发：

- 邻城空虚。
- 粮价异常。
- 某臣属可能倒戈。
- 地方缺水或劳力不足。
- 某人寻求赏赐、婚姻或职位。
- 雨季将至，战役窗口关闭。

界面显示理由而非全部公式，例如：“此人希望扩大自己家族的收入权”“目标守将与其有旧交”“估计敌援军 20 日内抵达”。

## 8. 经济、人口与劳役

### 核心资源

- 金钱：税收、市场、贸易；用于薪俸、外交、职业军和建设。
- 粮食：农业与贸易；用于人口、军队、城市和灾荒。
- 劳力：不是库存货币，而是可用家庭与时间；用于农务、建设、运输和征发。
- 政治能力：宫廷可同时推动的政策、任命和承诺容量，不无限囤积。
- 特殊装备：象、马、火器及其人员网络；数量少、来源具体。

### 人口组

第一版使用聚合家庭单位：

- 农业家庭。
- 城镇/工匠家庭。
- 流动与边缘人口。
- 武家/随从和职业军家庭。

征召农兵会减少当季劳力、延后人口恢复并增加逃亡/违抗风险。职业随从不抽离农务，但持续消耗现金和粮食。

### 税赋不是滑块免费收益

提高征收会影响：

- 实际服从和逃亡。
- 地方代理人的私留与腐败。
- 市场增长。
- 寺院或地方精英关系。
- 下一季生产和征兵意愿。

## 9. 战役、补给与战争

### 战役阶段

```text
目标和政治目的
→ 情报与外交准备
→ 集结地点和日期
→ 征发、采购、仓储和路线
→ 出征
→ 机动、会战、封锁或围城
→ 接受投降 / 撤军 / 继续追击
→ 战后统治安排
```

### 补给

军队可用补给来自：

- 随军携带。
- 友好仓库。
- 路线运输吞吐。
- 盟友/臣属承诺。
- 当地采购或征发。

当地征发是政治行为，会伤害服从、人口和长期产出；不能当作无代价自动补给。

### 战役计划界面

必须显示：

- 最早完成集结日。
- 预计安全作战天数。
- 关键运输瓶颈。
- 雨季/收获季距离。
- 敌方援军估计区间和情报年龄。
- 预计农务与财政代价。
- 无法执行的明确原因。

## 10. 会战与围城

### 会战目标设计

- 总兵数不是唯一答案。
- 战场以节点、退路、高地、渡口、本阵和夹击关系构成。
- 单位执行受统率、性格、体力、信息和命令延迟影响。
- 手操与自动结算必须使用同一战斗状态和命令模型。

第一版可先提供自动结算与可视化战报，待验证后增加节点式可暂停战斗。

### 围城

攻守双方同时面对多个时钟：

- 攻方粮食、疾病、士气和援军风险。
- 守方城粮、城防、水源、内部忠诚和民众压力。

选项包括封锁、强攻、破坏、谈判、收买、分兵迎援、撤围。城池投降可附带保留统治者、交人质、纳贡或驻军等条件。

## 11. 外交、情报与合法性

### 外交维度

- 信任：是否相信会履约。
- 亲近：人物与家族关系。
- 威胁：对扩张能力和意图的恐惧。
- 利益一致：共同敌人、贸易和安全。
- 地位：谁应向谁行礼、纳贡和承认。
- 历史债务：援助、背叛、未兑现承诺。

关系好不等于利益相同；关系差也不阻止短期联盟。

### 情报

每个势力维护独立知识状态：

- 估计值和区间。
- 取得日期。
- 来源与可信度。
- 是否可能被欺骗。

玩家不能直接看见所有真实兵力和忠诚。

### 合法性

合法性来自多个来源：

- 王族和继承规则。
- 军事胜利与保护能力。
- 宗教/寺院赞助和公正统治。
- 仪式性承认、贡使与象征物。
- 履行承诺、赏赐和仲裁。

它不是全局万能加成，而决定特定群体是否认为统治要求合理。

## 12. 胜利与失败

### 胜利路径

- **霸主胜利**：关键政治中心和主要统治者承认宗主，且关系稳定达到规定年限。
- **王权胜利**：合法性、礼仪承认、宗教赞助与王族延续达到目标。
- **贸易胜利**：控制关键通道、市场关系和港口收入网络。
- **统一胜利**：直接稳定控制核心河谷；最困难且非默认最优。
- **存续胜利**：小势力在强权夹缝中保持独立、繁荣和多代继承。

### 失败

- 王朝被彻底消灭且无可继承政治体。
- 宫廷失去全部可维持的权力中心。
- 剧本特定失败条件。

失去首都、被迫臣服或更换王朝不一定立即结束，可形成新的玩法状态。

## 13. 难度

不通过作弊式数值倍增作为主要难度：

- 情报模糊程度。
- 决策自动暂停和建议强度。
- AI 规划深度与安全边际。
- 玩家可撤销/预览范围。
- 初始政治结构和目标。

AI 获得的任何资源加成必须在高难度说明中明确列出，并且不能绕过核心规则。

## 14. 教程与可读性

教程围绕一个小型地方统治者展开，依次教授：

1. 读取实际控制与宗主关系。
2. 任命代理人。
3. 平衡劳力、粮食和征发。
4. 签订义务包。
5. 规划一个短途干季战役。
6. 处理投降与战后安排。

所有复杂结果显示“前三大影响因素”和至少一个可执行的改进建议。

## 15. 垂直切片内容

建议聚焦伊洛瓦底中下游：

- 4 个主要政治体。
- 12 个政治中心、约 48 个地区。
- 60 名人物。
- 一条河运主干、两条山地路线。
- 一次农业周期、一次干湿季转换。
- 完整任命、征发、战役、围城、臣属谈判和继承冲击。

它必须证明核心闭环，不能用大量事件、美术或角色数量掩盖系统无趣。
